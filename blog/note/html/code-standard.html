html编码规范

DivYang 2020-05-23 HTML

# 阿里编码规范HTML篇

# 项目命名

项目名全部小写，多个单词使用中划线分隔

正例：fisheep-blog，taobao

反例：fisheepBlog，fisheep_blog

# 目录命名

目录命名全部小写，用中划线分隔，有复数时使用复数，缩写不用复数

正例：scripts/utils/demo-scripts/doc/styles 反例：script/style/demo_scripts/demoStyles

# CSS、JS、SCSS、HTML、PNG等文件命名

同项目命名，项目名全部小写，多个单词使用中划线分隔

# 命名严谨性

代码中的命名严禁使用拼音和英文混合的方式，尽量使用正确的英文拼写和语法，避免歧义，易于理解。

# HTML规范

  • HTML5文档类型
  • 规定字符编码
  • IE兼容模式
  • doctype大写

正例：

    <!DOCTYPE html>
    <html lang="en">

    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <meta http-equiv="X-UA-Compatible" content="ie=edge">
      <title>标题</title>
    </head>

    <body>
    </body>

    </html>

# 缩进

使用2个空格缩进，节点嵌套需要缩进

# 分块注释

将代码各个部分用注释的方式标注出起止位置 正例：

    <!DOCTYPE html>
    <html lang="en">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <meta http-equiv="X-UA-Compatible" content="ie=edge">
      <title>标题</title>
    </head>

    <body>
       <!--header 头部 start-->
       <header>
         <nav></nav>
       </header>
       <!--header 头部 end-->
    </body>

    </html>
