【深入】参数值传递

DivYang 2020-10-30 JS

# 定义

ECMAScript中所有函数的参数都是按值传递的。

来自于 《JavaScript高级程序设计》

# 最直观的值传递

function plus2(num) {
  num = num + 2
  console.log(num)
}
let a = 1;
plus2(a) // 输出 3
console.log(a) // 输出 1

这没有什么可以解释的，最容易理解的值传递，当a传入plus2中，将a的值拷贝一份(假设为_a)给了num，改变_a的值，自然不会影响到原来的a。

# 不直观的值传递

function plus2(obj) {
  obj.value += 2
  console.log(obj.value)
}
let target = {
  value: 1,
}
plus2(target) // 输出 3
console.log(target) // 输出 3

按照网上大部分文章的解释: 当函数的参数为Object | Function时，传递方式为引用传递。 按照红宝书上面的解释: 传递Object | Function也是把引用拷贝一份给函数，即把target的引用(假设x0001)拷贝一份(假设x0002)给了obj。obj修改x0002下的value值，就会导致target的value值改变。

下面这个例子更容易理解。

function assign2(obj) {
  obj = 2
  console.log(obj)
}
let target = {
  value: 1,
}
assign2(target) // 输出 2
console.log(target.value) // 输出 1
