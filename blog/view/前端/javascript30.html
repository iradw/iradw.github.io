# javascript30个应用实战

# 第一天 键盘乐器

项目预览

# 思路

第一个项目较为简单，看一眼项目就能有大概思路

  1. 全局监听键盘事件onkeydown，按下对应的按键后播放音乐。
  2. 按键后切换屏幕上方块的样式，以表示键盘被按下。

HTMLMediaElement.currentTime 属性会以秒为单位返回当前媒体元素的播放时间。设置这个属性会改变媒体元素当前播放位置。

# 难点

  1. 连续按键导致音频不能播放，每次播放之前设置currentTime为0.
  2. 播放音频时，虚拟按键的激活样式应该只存在0.07s，如何在0.07s后恢复样式？给虚拟按键设置监听, transitionend过渡结束，和点击监听一样的写法
     item.addEventListener('transitionend', event => {
   if (event.propertyName === 'transform') { // 会有很多属性过渡结束，触发改时间，我们只需要transform的过渡
     item.classList.remove('play')
   }
 })

# 第二天 时钟

项目预览

# 思路

  1. 使用CSS绘制时钟，三个针的旋转可以使用transform: rotate(30deg)做
  2. 将当前时间的时分秒做对应旋转角度的转换

旋转时需要指定旋转中心transform-origin，默认的中心是div的中心，即50%, 50%位置，指针的旋转中心是指针的底部中心。

# 难点

  1. 如何将三个指针的旋转点重合，看下图

div的重叠都会在左上角重叠，当指针长短不一时，指针就会在上方重合，因此模拟时钟的指针排列，需要把指针的初始位置放在圆的下方

.hand {
  position: absolute;
  width: 6px;
  top: 50%; /* 将div顶部放在圆中间 */
  left: calc(50% - 3px);  /* 水平居中 */
  transform-origin: 50% 0;  /* 设置旋转中心为div顶部中心 */
}
