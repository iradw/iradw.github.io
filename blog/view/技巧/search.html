搜索引擎和Github搜索技巧

DivYang 2020-05-03

# 搜索引擎和Github搜索技巧

# github

# 命令in

in可以限定搜索范围，github仓库的项目分为3个部分，名称name，描述description和readme。

  • 在项目名中搜索: in:name 关键词
  • 在描述中搜索: in:description 关键词1 关键词2
  • 组合使用: in:description,readme 关键词1 关键词2

# 根据stars和forks数量搜索

  • 搜索react做的日历，且star在100以上的项目: in:description,readme react 日历 stars:>100
  • 搜索fork大于100的项目：forks:>100 不要忘记冒号

# language

某一种技术被多种语言实现，可以限定某一种语言使用language命令

  • 搜索用python写的验证码项目in:description 验证码 language:python

# 创建和更新日期

  • 搜索2020年1月1日以后创建的项目 created:>2020-01-01
  • 搜索2020年1月1日熠更新的项目 pushed:>2020-01-01 不要忘记冒号

# 搜索引擎

# 引号"" 完全匹配

不使用引号输入关键词的情况下，会出现分词现象，比如 如果加上了引号，就会进行精确匹配

# 减号

在搜索结果中去掉和某些词有关的内容，比如 ，图中的内容都是我们想要的结果，注：减号前有空格，后无空格。

# 星号

*是通配符，在两个关键词中添加，可以搜索到两个关键词中间包含其他任何内容的结果。

# site指令

这个是本人最常用的指令，可以在指定网站中搜索内容，这样就看不到*度的自家文章了。

# inurl指令

在url中搜索部分内容，有些网站的内容和域名相关性很高，可以精确查找网站，最好用英文搜索。

# intitle

在网页标题中搜索。

# intitle和inanchor

在锚点中搜索，也就是url中#后面的词，可以精确的搜索到文章中的标题。

文中的搜索引擎为dogedoge，称为国内的duckduckgo，不追踪不误导，搜索结果大多为比较专业的网站，可以完全放弃*度。
